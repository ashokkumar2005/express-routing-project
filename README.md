Basic Express.js routing project with multiple pages and custom 404 error handling.
# Express Routing Project 🚀

This is a basic Node.js and Express.js project demonstrating:

- Routing
- Redirect
- Serving HTML files
- Custom 404 error page
- Status codes

## 📁 Routes

- `/` → Home Page
- `/about` → About Page
- `/contact` → Contact Page
- `/index` → Redirect to Home
- Invalid route → 404 Page

## 🛠 Tech Used

- Node.js
- Express.js

## ▶ How To Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open:
   http://localhost:3000
